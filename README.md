# deep-learning-image-classification
Implementation of a Convolutional Neural Network (CNN) using TensorFlow/Keras to classify handwritten digits from the MNIST dataset with visualization of training performance.

## Objective

The objective of this task is to implement a **Deep Learning model for image classification** using TensorFlow. The model is trained on the MNIST dataset to classify handwritten digits.

---

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

---

## Dataset

The project uses the **MNIST Handwritten Digit Dataset**.

Dataset details:

* 60,000 training images
* 10,000 testing images
* Image size: 28 × 28 pixels
* Classes: Digits from **0 to 9**

---

## Model Architecture

A **Convolutional Neural Network (CNN)** is implemented with the following layers:

1. Convolution Layer (32 filters)
2. Max Pooling Layer
3. Convolution Layer (64 filters)
4. Max Pooling Layer
5. Flatten Layer
6. Dense Layer (64 neurons)
7. Output Layer (10 neurons with Softmax activation)

---

## Workflow

1. Load MNIST dataset
2. Normalize pixel values
3. Reshape images for CNN input
4. Build CNN model using TensorFlow/Keras
5. Train the model
6. Evaluate model performance
7. Visualize training and validation accuracy

---

## Results

The trained CNN model successfully classifies handwritten digits with high accuracy on the test dataset.

---

## Visualization

The project includes visualization of training accuracy and validation accuracy across epochs using Matplotlib.

---

## Conclusion

This project demonstrates how **Convolutional Neural Networks (CNNs)** can be applied to image classification tasks. Deep learning models like CNNs are widely used in computer vision applications.
